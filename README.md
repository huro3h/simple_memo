# Simple Memo

シンプルなメモ帳 Chrome 拡張機能です。

## 機能

- メモの作成・削除
- メモ一覧を新しい順で表示
- 作成日時の表示
- クリップボードへのコピー
- メモのエクスポート / インポート（JSON形式）

## インストール方法

### Chrome Web Store（準備中）

※ 公開後にリンクを追加予定

### 手動インストール（開発者向け）

1. このリポジトリをクローンまたはZIPでダウンロードする
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックする
5. ダウンロードしたフォルダを選択する

## 使い方

1. ツールバーの拡張機能アイコンをクリックしてポップアップを開く
2. テキストエリアにメモを入力し「追加」ボタンを押す（`Ctrl+Enter` / `Cmd+Enter` でも追加可能）
3. 各メモの「コピー」ボタンでクリップボードにコピーできる
4. 各メモの「削除」ボタンで削除できる

### Export / Import

- **Export JSON** — 現在のメモ一覧を `simple-memo-YYYY-MM-DD.json` としてダウンロードする
- **Import JSON** — エクスポートしたJSONファイルを読み込み、既存メモと結合する

## データについて

メモは端末内の `chrome.storage.local` にのみ保存されます。外部サーバーへの送信や第三者との共有は一切行いません。

## プライバシーポリシー

このアプリケーションが収集するデータはありません。入力されたメモはすべてお使いの端末内にのみ保存され、外部に送信されることはありません。
