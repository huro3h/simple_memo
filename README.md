# Simple Memo

シンプルなメモ帳 Chrome 拡張機能です。

<img height="400" alt="s" src="https://github.com/user-attachments/assets/57ebafac-c962-4deb-ac6a-da5500ccbacb" />


## 機能

- メモの作成・編集・削除
- メモ一覧を新しい順で表示
- クリップボードへのコピー
- メモにタイトルを追加
- メモのエクスポート / インポート（JSON形式）

## インストール方法

### Chrome Web Store（準備中）

※ 公開後にリンクを追加予定

### 手動インストール（開発者向け）

1. このリポジトリをクローンまたはZIPでダウンロードする
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックする
5. ダウンロードしたフォルダを選択する

## 使い方

1. ツールバーの拡張機能アイコンをクリックしてポップアップを開く
2. テキストエリアにメモを入力し、＋ボタンを押す（`Ctrl+Enter` / `Cmd+Enter` でも追加可能）
3. 各メモのタイトル部分をクリックするとタイトルを編集できる（`Enter` で確定、`Escape` でキャンセル）
4. 各メモのアイコンボタンで以下の操作ができる：
   - **コピー**（重なった四角アイコン）— クリップボードにコピーする
   - **編集**（鉛筆アイコン）— メモ本文を編集する。チェックマークで保存、✕ でキャンセル
   - **削除**（ゴミ箱アイコン）— メモを削除する

### Export / Import

- **Export JSON** — 現在のメモ一覧を `simple-memo-YYYY-MM-DD.json` としてダウンロードする
- **Import JSON** — エクスポートしたJSONファイルを読み込み、既存メモと結合する

## データについて

メモは端末内の `chrome.storage.local` にのみ保存されます。外部サーバーへの送信や第三者との共有は一切行いません。

## プライバシーポリシー

このアプリケーションが収集するデータはありません。入力されたメモはすべてお使いの端末内にのみ保存され、外部に送信されることはありません。
